== Description ==
The Plugin add a column to the orders list in the admin interface to filter and sort orders by payment method

== Installation ==
Just install by wordpress plugin directory or download the plugin and upload to your wordpress site

== Screenshots ==
1. The new column with payment method sortable and filtrable
2. Different icon for different payment method in order list
3. Settings page where you cam choose different icon for each payment method
